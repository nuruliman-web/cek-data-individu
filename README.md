# 📋 Aplikasi Cek Validasi Data Individu

Aplikasi berbasis Streamlit untuk mengecek kelengkapan data Excel pada kolom D sampai Q.

## 🎯 Fitur
- Upload file Excel (.xlsx)
- Menampilkan header/kolom yang tersedia
- Mengecek data kosong di kolom D-Q
- Logika khusus: BO wajib diisi jika Pekerjaan = "PELAJAR/MAHASISWA" atau "IBU RUMAH TANGGA"
- Menampilkan tabel ceklis per baris
- Filter data bermasalah
- Download hasil pengecekan
